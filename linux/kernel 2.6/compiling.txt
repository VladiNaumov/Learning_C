
/*************  компиляция  ядра Linux 2.6 на Ubuntu 10.04 *****************************************************/

Для компиляции ядра Linux 2.6 на Ubuntu 10.04 вам понадобятся определённые пакеты и инструменты.
Вот как можно проверить наличие необходимых пакетов и установить их, если они отсутствуют:

Необходимые пакеты:
Компилятор GCC — для компиляции исходного кода.
Make — для управления процессом сборки.
libncurses5-dev — для текстового интерфейса конфигурации ядра.
bison — генератор синтаксических анализаторов.
flex — генератор лексических анализаторов.
module-init-tools — для работы с модулями ядра.
kernel-package — утилиты для создания пакетов ядра (опционально).
patch — для применения патчей к исходным кодам.

Вы можете проверить, установлены ли пакеты, с помощью команд:

Для компиляции ядра на Ubuntu 10.04 вам понадобятся следующие пакеты:

*build-essential - включает компилятор gcc, g++, make, и другие основные инструменты для компиляции:
bash
sudo apt-get install build-essential

libncurses5-dev - для поддержки текстового интерфейса при конфигурации ядра:
bash
sudo apt-get install libncurses5-dev

*fakeroot - позволяет создавать root-права для создания пакетов Debian:
bash
sudo apt-get install fakeroot


*kernel-package - для создания пакета ядра:
bash
sudo apt-get install kernel-package


*bzip2 - для распаковки сжатых файлов:
bash
sudo apt-get install bzip2


*flex и *bison - инструменты для генерации лексических и синтаксических анализаторов:
bash
sudo apt-get install flex bison


*module-init-tools - для работы с модулями ядра:
bash
sudo apt-get install module-init-tools


libssl-dev - библиотеки для работы с SSL:
bash
sudo apt-get install libssl-dev


git-core (если ядро нужно скачать из репозитория git):
bash
sudo apt-get install git-core


Команды для проверки наличия пакетов в системе:
Проверить, установлен ли пакет, можно с помощью команды:

bash
dpkg -l | grep <имя_пакета>
Например:

bash
Копировать код
dpkg -l | grep build-essential
dpkg -l | grep libncurses5-dev
dpkg -l | grep fakeroot
dpkg -l | grep kernel-package
dpkg -l | grep bzip2
dpkg -l | grep flex
dpkg -l | grep bison
dpkg -l | grep module-init-tools
dpkg -l | grep libssl-dev
dpkg -l | grep git-core

Эти команды выведут информацию о состоянии пакета, если он установлен.

Нет пароля для root: При установке Ubuntu, вы создаете обычного пользователя, который получает права для выполнения административных команд через sudo.
Это означает, что у пользователя, созданного при установке, нет прямого пароля для root.

Использование sudo: Вы можете выполнять команды с привилегиями root, используя sudo. Например:

sudo apt-get update
При этом вам нужно ввести пароль вашего текущего пользователя (не root).

Как установить пароль для root:
Если вам необходимо установить или изменить пароль для root, чтобы иметь возможность использовать su, выполните следующие шаги:

Получите права суперпользователя с помощью sudo:

sudo -i
Введите пароль вашего пользователя.

Установите пароль для root с помощью команды passwd:

passwd
Вас попросят ввести новый пароль для root. Введите его дважды для подтверждения.

Для установки пакетов с USB флешки на Ubuntu 10.04 выполните следующие шаги:

Подключите USB флешку: Подключите флешку к компьютеру. Обычно она монтируется автоматически в директорию /media.

Проверьте точку монтирования: Узнайте, где именно была смонтирована флешка, выполнив команду:

bash
df -h
Вы увидите список всех смонтированных файловых систем. Найдите в списке вашу флешку, она будет указана как /dev/sdX1, где X — это буква, соответствующая вашей флешке. Точка монтирования будет указана в последней колонке, например, /media/usb.

Перейдите в директорию с пакетами: Перейдите в директорию на флешке, где находятся пакеты:

bash
cd /media/usb/<путь_к_пакетам>
Установите пакеты с помощью dpkg: Для установки пакетов используйте следующую команду:

bash
sudo dpkg -i *.deb
Эта команда установит все .deb пакеты, находящиеся в текущей директории.

Установите недостающие зависимости (если есть): После установки пакетов могут потребоваться дополнительные зависимости. Вы можете установить их с помощью команды:

bash
sudo apt-get install -f

Эти шаги позволят установить пакеты с флешки на вашу систему.

/************************************************************************************************************************************/

Настройте и соберите ядро: Перейдите в каталог с исходным кодом и выполните:

bash
make menuconfig  # или make xconfig, если у вас установлен xconfig
make
make modules
Установите новое ядро: После сборки установите его в систему и обновите загрузчик (например, GRUB) для использования нового ядра.
